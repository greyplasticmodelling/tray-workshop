Tray Workshop guide

Free wargaming movement tray STL generator

Tray Workshop creates browser-generated STL files for tabletop movement trays. It is designed for players who want to test tray dimensions before sending a file to a 3D printer.

Build a movement tray STL

Useful tray options

  • Standard rank-and-file movement trays with selectable front, rear, left, and right rails.
  • Lance wedge trays for formations that widen by rank.
  • Adapter trays for fitting one base footprint into another.
  • Optional magnet cutouts, generated matching bases, and advanced tray finish options.

Unofficial fan-made utility for tabletop wargaming. Check your game system and event rules before printing.
