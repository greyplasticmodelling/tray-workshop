Tray Workshop guide

Circle and oval base adapter trays

Some tabletop units use round or oval bases but still need clean movement trays, display trays, or adapter layouts. Tray Workshop includes circle adapter trays and fixed oval adapter presets.

Open the tray generator

When to use these tray types

  • Use circle adapter trays for ranked round bases with a controlled gap between cutouts.
  • Use oval adapter trays for fixed oval base presets such as 60 x 35 mm, 75 x 42 mm, and 90 x 52 mm.
  • Use border controls to add extra material around the outside of the tray for handling or decoration.

The oval presets include built-in allowances for the shape of common plastic gaming bases.
